## What Is In Each Folder

- exp.ipynb: Quick experiments and rough notebook work.
- deep_leaning/ANN: Simple artificial neural network practice. Includes notebook, student score data, saved model weights, and graph image.
- deep_leaning/CNN: CNN notebook practice.
- deep_leaning/GRU: GRU text-related work with processed data, tag file, notebook, and generated wordcloud images.
- deep_leaning/LSTM: LSTM spam-detection style practice with dataset and notebooks.
- deep_leaning/RNN: RNN sentiment work using IMDB dataset and related notebooks.
- NLP: NLP task notebooks, processed files, trained model output, confusion matrix image, and wordcloud outputs.
- NLP/harry_text: Harry Potter text files (Book 1 to Book 7) used as raw text data.
